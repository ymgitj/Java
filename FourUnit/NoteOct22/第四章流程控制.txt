一"复合语句
以整个块为单位的语句又称块语句。
以开括号{开始.闭括号}结束
复合语句为局部变量提供一个作用，该作用域作为程序的
一部分，在该作用域某个变量被创建并能使用。
二"条件语句
1.简单的if语句
if(布尔表达式){
   语句序列；
}
2.if...else语句
if(布尔表达式){
    语句序列;
}
else{
   语句序列;
}
3.if...else if...else if
if(表达式1){
    若干语句
}
else if(表达式2){
    若干语句
}
else if(表达式){
    若干语句
}
//if 语句只执行条件为真的语句
三"switch分支语句
switch(表达式){
     case 常量值1：
          语句块1;
          [break;]
     case 常量值2:
          语句块2;
          [break;]
     default 常量值：
          [break;]
 }
 //同一个switch语句,case的常量值必需互不相同
 //case 后面的表达式可以为整数，但不可以是实数
 四"循环语句
 1.while循环语句
 while(条件表达式)
 {
   执行语句
 }
 //条件表达式为真执行{}中的语句，当执行完{}后重新判断
 条件表达式的返回值，直到返回的值为假，结束退出循环。
 //while()后面加分号无报错但影响调试时间
 2.do while()循环语句
 do{
   执行语句
 }while(条件表达式);
 //与while()循环语句明显区别do while()循环语句while()后面多了个分号
 3.for循环语句
 for(表达式1;表达式2;表达式){
     语句序列
  }
  //表达式1：初始化表达式，负责变量的初始化。
  //表达式2：循环条件表达式，指定循环条件。
  //表达式3：负责修整变量，改变循环条件。
4.foreach语句
for(元素变量x:遍历对象 obj){
     引用了x的语句;
 }
