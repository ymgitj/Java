一>成员变量
//1.在类体中定义的变量为成员变量
//2.类的成员变量又可以分为"静态变量"和"实例变量"
class var{
   int age=20; //定义实例变量
   static int weight=50;//定义静态变量
}
二>局部变量
1.在类的方法体中定义的变量(即 { } 之间的变量)称之为局部变量
2.局部变量的生命周期取决于方法，该方法被调用时，java虚拟机为其分配内存空间，当该方法结束时会释放该局部变量占用的内存空间

//局部变量的名字可与成员变量名称相同…此时成员变量将被隐藏…即这个成员变量在此方法中暂时失效…如果想调用…需要使用 "类名.静态变量"  调用


//成员变量在整个类中都有效，局部变量只在方法体中有效，
